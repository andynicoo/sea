COMPANY REGISTRATION COMMON QA
Q1、企业出海注册当地公司主要涉及到哪些服务?

近年来,越来越多的中国企业“出海”寻找商机，注册海外公司也逐渐成为了一个热门话题。同时，海外公司注册后涉及到的后续服务也越来越多。 归纳起来，企业出海注册当地公司主要涉及四方面的内容:

1.注册公司:包含公司注册,公认证,董事委托，注销，名称、股东、董事、经营范围、注册股本变更等

2.银行开户:比如本地银行、离岸银行、网络银行开户等

3.地址租赁:比如秘书地址挂靠、联合办公空间租赁、独立办公室租赁等

4.年申报税:年审，记账报税，审计,销售税许可证申请

Q2、香港公司注册后,怎么选择银行?

很多企业开设香港公司之后，都会面临银行开户的选择。银行开户跟企业是一个双向选择，不单纯是费用的问题，也需要双方都有意向。就像男女相亲，需要彼此看对眼才能促成。香港银行开户的关键与核心在于,证明业务的真实性与开户的必要性。所谓的真实性就是企业需要提供关联公司的资料，开户的必要性 是指需要有海外的业务合同。因此银行的基本推荐原则如下:

1.年营业额过亿,推荐汇丰银行、中国银行、花旗银行;

2.可提供国内关联公司完整配套的2~3份业务&采购合同，推荐汇丰/东亚银行;

3.可提供香港公司与国内关联公司的完整配套采购/贸易资料,推荐离岸汇丰/稠州银行;

4.可提供真实存在的美国供应商/客户合同，推荐华美银行;

5.以上都不满足,那就 CBI,泛付,以及其他的第三方支付工具。

Q3、香港公司注册所需哪些资料?

部分企业有了注册公司的意向后，会顾虑注册公司时所需的资料比较繁琐，实际上所需准备的资料比较简单,一般所需的参考资料如下:

1.拟定公司中英文名称(如香港xx有限公司，HONGKONG*xCO，LIMITED);

2.确定公司业务性质(即经营范围,如电子产品,元器件销售);

3.注册资本(默认1万港币，无需验资);

4.所有董事股东的身份证或者护照扫描件;

5.所有董事和股东手持身份证照片;

6.股东以及相对应的股份比例分配

7.董事及股东的签名。

Q4、注册美国公司后,需要年审吗?

美国公司注册后，为了让州政府了解公司的运营情况，需要年审。年审根据各州政府规定进行。年审完成后,公司状态保持正常。

Q5、美国公司年审后,需要报税吗?

年审是向注册州政府汇报公司状态，以便保证公司具备继续经营的资格。报税一般是指纳税主体向政府 报收入,缴纳相应的税费，两者是完全不同的。所以注册了美国公司之后，每年除了年审，还要报税，不然 惹上麻烦。

Q6、美国公司注册中，LLC公司和INC公司有什么差异?

LLC的含义是有限责任公司，INC代表了股份有限公司，目前大多数海外人士注册的美国公司是INC(股份有限公司),它也是中国籍法人注册类型最多的美国公司。

Q7、塞浦路斯公司很热门,具体注册优势体现在哪些方面?

塞浦路斯公司的主要优势体现在以下方面:

1.欧盟公司VAT不代扣代缴;

2.提供中国大陆居民身份证即可办理公司注册;

3.欧盟成员国间互免关税;

4.企业所得税和增值税为欧洲最低,维护成本低;

5.可自由在境内外开设银行账号

Q8、注册法国公司,有哪些优势??

1.注册本土公司自动下发VAT;

2.清关时自动递延，缓解现金流压力;

3.亚马逊平台欧盟VAT不代扣代缴;

4.可入驻法国电商平台 Amazon、Cdiscount、Veepee、Booking.com、Fnac.com 等

5.国际法律地位高,有利于迅速提升企业及其产品的品牌形象

6.利于开立境外银行账户方便资金流转。
