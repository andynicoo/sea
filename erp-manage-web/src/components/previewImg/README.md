## API

previewImg(config)

| 属性   | 说明                                   | 类型                | 默认值 |
| ------ | -------------------------------------- | ------------------- | ------ |
| config | 可以是对象也可是一个字符串,图片src地址 | string\|object(url) | -      |
|        |                                        |                     |        |

```js
this.$previewImg({url:""})或this.$previewImg(url)
```
