图片配置属性说明

属性 是否必填 类型 说明
angle 否 number 图片旋转角度
height 否 number 图片高度，如果填写高度小于图片真实高度则图片会被裁剪
left 否 number left距离
opacity 否 number 透明度
originX 否 string 对象变换的水平原点，可能值为"left", "right", "center"
originY 否 string 对象变换的垂直原点，可能值为"top", "bottom", "center"
src 是 string 图片的src
top 否 number top距离
type 是 string 必须填写：image
scaleX 否 number 图片水平方向缩放大小
scaleY 否 number 图片垂直方向缩放大小
width 否 number 图片宽度，如果填写宽度小于图片真实宽度则图片会被裁剪

文字配置属性说明

属性 是否必填 类型 说明
angle 否 number 旋转角度
fill 否 string 颜色
fontFamily 否 string 字体
fontSize 否 number 字体大小
fontStyle 否 string 字体样式，可能的值为"normal", "italic", "oblique"
fontWeight 否 string或number 字体粗细，可能的值为bold, normal, 400, 600, 800
height 否 number 字体高度
left 否 number left距离
opacity 否 number 透明度
originX 否 string 对象变换的水平原点，可能值为"left", "right", "center"
originY 否 string 对象变换的垂直原点，可能值为"top", "bottom", "center"
scaleX 否 number 水平方向缩放大小
scaleY 否 number 垂直方向缩放大小
text 是 string 文本内容
textAlign 否 string 文字对齐方式，可能的值为"left", "center", "right", "justify", "justify-left", "justify-center" or "justify-right"
top 否 number top距离
type 是 string 必须填写：textbox
width 否 number 字体宽度

!!#ff0000 注意事项说明!!

在配置的时候需要一个背景层，这个层的id为workarea，其宽高决定了整个画布显示的宽高可以设置其top与left值为0，这样当其余元素的top、left为0的时候，就会呈现出一致的位置效果

// 配置事例
[
  {
    id: 'workarea',
    top: 0,
    left: 0,
    width: 1242,
    height: 2208,
  },
  {
    type: 'image',
    top: 0,
    left: 0,
    src: 'https://static.taishan.qq.com/editor/assets/2997b0c920be29c12c16ae3a2b2bc378.png',
  },
]
